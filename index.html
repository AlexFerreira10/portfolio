Alguns dos meus projetos

Conecta Uerj

Atuo no Conecta UERJ como desenvolvedor Java backend com Postgres. Este é um projeto de pró-docência da UERJ que gerencia eventos, palestras e projetos, desde a inscrição até a emissão de certificados, além de divulgar novidades da universidade e registrar atividades complementares dos alunos.

Facilita Uerj

Atuo desde a modelagem do sistema e definição de regras de negócio, além de trabalhar como desenvolvedor fullstack com Java, Angular e MySQL. O Facilita UERJ é um projeto voluntário que busca melhorar a comunicação entre alunos e coordenadores, oferecendo agendamento de reuniões e integração com a API do Google Calendar para facilitar a organização dos coordenadores.

Investor Bee

Esta aplicação backend é um agregador de investimentos para gerenciar ações na bolsa de valores. Utiliza a API da Brapi via OpenFeign para obter dados e fornece CRUDs para gerenciar a carteira de investimentos. Inclui conteinerização com Docker, padronização de API RESTful, segurança com JWT, documentação via Swagger, testes unitários com JUnit e Mockito, e persistência de dados em MySQL.

API para Controle de Produtos

Esta aplicação fornece uma API para cadastrar produtos e fornecedores em uma loja. Ela notifica os fornecedores via e-mail sobre a chegada dos produtos. Implementa padronização de API REST, autenticação com JWT, envio de e-mails personalizados com Thymeleaf e JavaMailSender, além de documentação com Swagger e persistência de dados no MySQL.

Pesquisador de Palavras em Arquivos com Árvore AVL

Este projeto implementa uma árvore AVL em Java para pesquisa de palavras em coleções de arquivos de texto. Facilita a busca por palavras específicas, fornecendo informações sobre sua frequência e ocorrência.

API para Controle de Medicamentos

Esta aplicação fornece uma API para o gerenciamento de remédios. Neste projeto implemento mecanismos de segurança via JWT Token, CRUD e persistência de dados com JPA/Hibernate e MySQL.

Atualmente, estou em busca de uma oportunidade de estágio em ambas as áreas, para que eu possa continuar aprendendo e aprimorando meus conhecimentos no campo da tecnologia.

Ao longo da minha trajetória, também tive contato com diversas outras tecnologias, como C, C++, JavaScript, HTML, CSS e Angular, que ampliaram minha compreensão de conceitos fundamentais e me proporcionaram uma base sólida para meu desenvolvimento profissional.
